      Array的使用：「能用Array 就不要用List」   
今有一個List 紀錄美元的價格，   
要換算成台幣價格時，   
可以將List 直接轉成 Array，   
會比用for迴圈轉快很多   




```python
%matplotlib inline
import matplotlib.pyplot as plt
import numpy as np
```


```python
L = [10,20,30]
```


```python
e = 27.5
```


```python
price = np.array(L)
```


```python
price = price*e
```


```python
price
```




    array([275., 550., 825.])



也可以轉小數


```python
M = [12.3,23.4,34.5]
```


```python
price_M = np.array(M)
```


```python
price_M = price_M*e
```


```python
price_M
```

```python
score = np.array([90,80,95])
weight = np.array([.3,.4,.3])
```

用 Array 算成績，快速版


```python
np.dot(score,weight)
```
